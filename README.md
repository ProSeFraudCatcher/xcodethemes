### Installation

#### Theme Installer
1. Find and install [ThemeInstaller][1] app from the Mac App Store. 
2. Open any theme you want to install and it's done. Themes are visible once Xcode is restarted.

#### Manually 
1. Copy files you want to install
2. Go to `~/Library/Developer/Xcode/UserData/`
3. Paste copied files
4. Restart Xcode

[1]:	https://itunes.apple.com/app/themeinstaller/id1148208665
